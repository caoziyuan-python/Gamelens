# Prompt模板-项目内容打标

这个模板用于统一管理“可复用正文”和“项目专属信息”。

核心原则：

1. 整个文件默认都可复用
2. 只有必须跟项目变化的内容才打标
3. 统一使用 `【需按项目替换：...】`
4. 换项目时只搜索 `需按项目替换`

## 推荐标记

- `【需按项目替换：项目名称】`
- `【需按项目替换：文件名】`
- `【需按项目替换：项目背景】`
- `【需按项目替换：字段名】`
- `【需按项目替换：业务目标】`
- `【需按项目替换：粘贴代码】`

## 标准结构

```md
# 主功能-关键限制-适用对象

## 用途
一句话说明这份 prompt 的用途

## 使用说明
- 只修改带有 `【需按项目替换】` 的部分
- 其他内容默认直接复用

## Prompt正文

你是一名……

任务：
请你……

补充信息：
- 项目名称：`【需按项目替换：项目名称】`
- 当前文件：`【需按项目替换：文件名】`
- 项目背景：`【需按项目替换：项目背景，可不填】`
- 业务目标：`【需按项目替换：业务目标，可不填】`

输出要求：
1. ……
2. ……
3. ……
```

## 不建议写法

- `[项目名称]`
- `{{项目名称}}`
- `<项目名称>`

这些写法不够统一，也不方便后期全文搜索替换。
