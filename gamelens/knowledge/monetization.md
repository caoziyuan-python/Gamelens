# 移动游戏变现策略知识库
# 来源：Unity Mobile Gaming Report / Sensor Tower / Adjust行业基准数据整理

## 广告变现核心策略

### 激励式广告 Rewarded Video Ads
- 定义：用户主动选择观看完整广告以获取游戏内奖励
- 行业完播率：82-88%（远高于插屏广告的60-70%）
- eCPM范围：$8-25（美国市场），$2-8（东南亚市场）
- 用户满意度：Net Promoter Score比强制广告高40分以上
- 最佳触发场景：
  关卡失败后（转化率最高，约35%点击）
  道具或提示不足时
  每日奖励入口（固定位置，用户主动寻找）
  复活机会（关键时刻，付费意愿最强）
- 频次上限：每会话不超过5次，避免用户产生疲劳感
- 关键设计：奖励价值感知要高于实际成本，让用户觉得「赚到了」

### 插屏广告 Interstitial Ads
- 触发频次红线：每3-5个关卡最多1次
- 超频后果：次日留存下降12-18%，用户评分平均下降0.3-0.5星
- 最佳插入点：关卡结算界面、地图切换时的自然停顿
- 时长建议：15秒以内；超过30秒跳出率超过40%
- 绝对禁止：游戏核心玩法进行中、教程阶段、付费用户
- 与激励式广告组合：插屏负责保底收益，激励式负责用户体验

### 横幅广告 Banner Ads
- 适用页面：主菜单、商店、排行榜、成就页
- 不适用：任何游戏内核心玩法界面
- 点击率（CTR）：0.1-0.3%，显著低于视频广告
- 收益：约为激励视频的15-20%
- 建议用途：作为补充收益，不作为主要变现手段

### 去广告订阅 Ad-Free Subscription
- 转化率基准：活跃用户中5-12%愿意付费去广告
- 定价甜蜜点：月付6-15元人民币（$1-2美元）
- 关键发现：去广告用户的30日留存比普通用户高25-35%
- 触发时机：用户连续遭遇3次以上广告后弹出订阅引导转化率最高
- 捆绑策略：去广告+专属皮肤+每日金币，比单纯去广告转化高40%

## 付费转化漏斗设计

### 标准转化路径
新增用户 → D1活跃 → D7回访 → 首次付费意愿 → 首次付费 → 复购

### 各阶段转化率行业基准
- D1活跃率：45-55%（良好），35%以下需优化新手引导
- 付费意愿触达率：约20-25%活跃用户会接触到付费点
- 首次付费转化：2-5%总安装用户（休闲游戏偏低端）
- 复购率：首次付费用户中38-45%会在30天内复购
- 平均付费用户ARPU：$3-8/月（休闲游戏），$15-30/月（中重度）

### 付费点设计原则
- 挫败感触发：关卡失败3次后出现付费引导，转化率最高
- 限时优惠：首次付费弹出「首充特惠」，折扣感知提升转化50%
- 小额起步：$0.99首充比$4.99首充转化率高3倍，但LTV相近
- 社交压力：好友排行榜附近出现付费入口，转化提升15-20%

## 变现模式组合建议

### 休闲游戏推荐组合（按用户分层）
- 免费轻度用户：插屏广告 + 横幅广告（保底收益）
- 活跃用户：激励式广告为主（高eCPM + 好体验）
- 重度用户：去广告订阅 + 道具内购（最高LTV）

### 变现健康度警戒线
- 广告收益占总收益超过85%：过度依赖广告，抗风险能力弱
- 付费用户占比低于1%：付费体系设计有问题
- 激励式广告点击率低于15%：奖励设计吸引力不足
